API接口文档

1. 获取国家列表

GET /api/countries

返回示例：["美国", "中国", "加拿大", "英国", "日本"]

2. 获取项目列表

GET /api/projects

返回示例：["Telegram", "WhatsApp", "微信", "支付宝", "抖音"]

3. 获取线路列表

GET /api/lines?country=美国&project=Telegram

返回示例：["线路1", "线路2"]

4. 下单购买

POST /api/buy

参数示例：{"country": "美国", "project": "Telegram", "line": "线路1"}

5. 获取订单状态

GET /api/order-status?id=123456

返回示例：{"status": "等待中", "sms": ""}
